# Research & Reuse

Mandatory before any new implementation:

1. **GitHub code search first:** `gh search repos` and `gh search code` for existing implementations before writing anything new.
2. **Library docs second:** Use Context7 or primary vendor docs to confirm API behavior and version-specific details.
3. **Check package registries:** Search npm, PyPI, crates.io before writing utility code. Prefer battle-tested libraries.
4. **Adopt proven approaches** over net-new code when they meet the requirement.
